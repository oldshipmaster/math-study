# Lessons 3–5 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增「数列与规律」「数列与图形」「正数与负数」三节完整互动动画课并发布。

**Architecture:** 扩展课程强类型数据并把三种专属互动场景拆到独立组件文件，由现有 `Scene` 分派器和播放器复用。课程进度仍按课程 ID 存入同一 localStorage 结构，旧数据天然兼容。

**Tech Stack:** React 19、TypeScript、CSS animations、Vitest、Testing Library、vinext、GitHub Pages

## Global Constraints

- 面向 7–12 岁，保持编辑感数学实验室视觉。
- 新增 3 节课，每节 6 个场景，最后场景为 3 题测验。
- 支持鼠标、触屏、键盘按钮和减少动态效果。
- 不复制 B 站视频画面、配音或逐句文案。
- 旧的第 1、2 课进度与重置功能必须保持兼容。
