# 第 61–65 课设计

1. 整除性检验：用末位、数位和、末两位和末三位的侦探卡判断 2、3、4、5、6、8、9、10、12 的整除性。
2. 分块除法：147=140+7，分别除以 7，再把商 20+1 合并。
3. 使用因数对做除法：把 12 的因数对翻成除法事实，并延伸到 120÷30、150÷50；明确 0 不能作除数。
4. 扩展短除法：156÷7 依次减 70、70、14，记录 10+10+2，得到 22 余 2。
5. 短除法：将上述重复减法压缩为逐位“除、写商、带余数”的竖式过程。

每课 5 个 HTML/CSS 动画场景和一个三题测验，延续即时反馈、键盘/触屏、进度保存与按课重置。
