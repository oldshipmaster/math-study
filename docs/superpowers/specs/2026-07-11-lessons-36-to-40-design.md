# 第 36–40 课互动课程设计

日期：2026-07-11

新增标准竖式加法、减法口诀、分块减法、店主补差法、扩展竖式减法。

- 标准竖式加法：数位对齐，从个位起算，满十向前进一；覆盖整数和小数。
- 减法口诀：用凑十加法事实反推减法，理解减法是加法逆运算。
- 分块减法：把减数按十位和个位拆开，连续减去。
- 店主补差法：从价格逐段补到整角、整元、付款额，再合计补数。
- 扩展竖式减法：把差理解为从小数补到大数的距离，按个位、整十、整百分段补齐。

标准竖式加法扩展 `AdditionScene`；新增 `SubtractionScene` 处理口诀、分块和补差。课程总数 40，支持独立进度重置并完成测试、构建和发布验证。
