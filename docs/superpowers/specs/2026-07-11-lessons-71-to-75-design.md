# 第 71–75 课设计

1. 算术法则：交换律、结合律、分配律，用 5+6、6×15、3×14 做动画变形。
2. 使用计算器：估算 307×49≈15000，再练习数字、小数点、正负号、四则、百分比和平方根按键。
3. 长度：认识长度/高度/宽度和 km、m、cm、mm 的千百十进制转换。
4. 长度计算：先统一单位，再计算树木生长与 760m+1.2km+630m=2.59km。
5. 周长：沿边走一圈求和，并用对边关系补出凹形未知边，得到 46m。

每课包含 5 个 HTML/CSS 动画互动场景和一个三题测验；新增计算器与测量实验台，保留触屏、键盘、即时反馈、进度保存和按课重置。
